Model

The model is the data which is merged with the template to produce the view.

lure.js

				
			/* Globals */
			var controllers = {}; 
			var routes = {};
			var scope = {};
			var thisRoute;

The model is how you structure your data. You will create a hierarchy in the global scope object, so that it will merge nicely with your templates.

Scope is a global variable that all controllers have access to. This is how data is shared throughout your app. You could say that the scope is equivalent to the model.

lure.js


			// store model
			var model = scope;

After the model has been set it will be merged with a template to create our next topic the View.
