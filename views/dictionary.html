dictionary

definitions for all those buzz words

Route

Routes make it easy to wire together the current URL location, view templates, and controllers.

Template

Template is declarative html that, along with information from the model, becomes the view that a user sees in the browser.

Model

The model is the data which is merged with the template to produce the view.

View

The view begins its life as a template, is merged with the model, and finally rendered into the browser DOM.

Controller

It's job is to construct the model and publish it to the view.
