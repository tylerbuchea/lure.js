Model

The model is the data which is merged with the template to produce the view.

components/lure/lure.js

				
			/* Globals */
			var controllers = {}; 
			var model = {};
			var routes = {};
			var thisRoute;

The Model is how you structure your data. You will create a hierarchy in the global model object, so that it will merge nicely with your templates.

Model is a global variable that all controllers have access to. This is how data is shared throughout your app.

The Model will be merged with a template to create our next topic the View.
