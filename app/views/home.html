What is MVC?

Model

The model is the data which is merged with the template to produce the view.

View

The view begins its life as a template, is merged with the model, and finally rendered into the browser DOM.

Controller

It’s job is to construct the model and publish it to the view.

What is Lure.js?

It is a tiny educational MVC framework written in JavaScript. What does that mean? It is an MVC framework written for JavaScript programmers that want to learn about MVC.

The code is robust enough to be useful (in fact this site is using Lure.js). But small enough (~120 lines) for even a novice coder to read through and understand not only how an MVC is implemented but how a framework is constructed.
