View

The view begins its life as a template, is merged with the model, and finally rendered into the browser DOM.

lure.js

	
			// store model
			var model = scope;

			// prepare template
			var template = Handlebars.compile(html);

			// create view from model & template
			var view = template(model);

The view is created when the model-data (in our case the scope object) is merged with a template.

Lure.js uses Handlebars for template interpolation. Handlebars is a very easy framework and you can check out the API reference here Handlebars API.
